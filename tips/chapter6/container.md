## Container

Docker는 클라이언트/서버 구조
- 도커 데몬은 모든 리소스(이미지, 네트워크, 컨테이너)를 관리
- 컨테이너 저장소와도 연결해 이미지 업로드/다운로드 가능

컨테이너 이미지=경량 실행 파일 패키지

### 컨테이너 이미지
- 셸 명령어는 베이스 이미지로 사용하는 리눅스 배포판에 따라 다르다.
  - Ubuntu, Debian
  - Alpine
  - Centos, RHEL, Fedora

```shell
docker login ghcr.io
```

컨테이너 이미지로 패키징
- 고립된 컨텍스트에서 계산 소스와 네트워크를 가지고 애플리케이션을 실행한다는 것

- 고립된 컨텍스트는 컴퓨터 안에 만들어진 완벽하게 독립적인 가상의 방
- 계산 소스는 컴퓨팅 자원(CPU/RAM)
- 네트워크는 자기만의 고유한 IP주소와 통신포트를 독립적으로 가진다는 뜻
